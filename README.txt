=========================================
   ASHLEY INVENTORY SYSTEM - HƯỚNG DẪN
=========================================

📁 FILES TRONG THỦ MỤC:
- START_ASHLEY_INVENTORY.bat     : Khởi động hệ thống
- CREATE_DESKTOP_SHORTCUT.bat   : Tạo shortcut trên desktop
- OPEN_FROM_ANOTHER_COMPUTER.html: Mở từ máy tính khác
- server.js                     : Server chính
- index.html                    : Giao diện web
- get-data.ps1                  : Script PowerShell

🚀 CÁCH SỬ DỤNG:

1. KHỞI ĐỘNG HỆ THỐNG:
   - Double-click "START_ASHLEY_INVENTORY.bat"
   - Đợi browser tự động mở
   - Hoặc mở browser và vào: http://localhost:3001

2. TẠO SHORTCUT (Tùy chọn):
   - Double-click "CREATE_DESKTOP_SHORTCUT.bat"
   - Shortcut sẽ xuất hiện trên Desktop

3. TRUY CẬP TỪ MÁY KHÁC:
   - Mở "OPEN_FROM_ANOTHER_COMPUTER.html"
   - Click các link để thử kết nối
   - Hoặc dùng IP của máy chủ: http://[IP]:3001

🔧 TROUBLESHOOTING:

- Nếu lỗi "Node.js not found": Cài Node.js hoặc dùng portable version
- Nếu không kết nối được từ máy khác: Kiểm tra firewall
- Nếu port 3001 bị chiếm: Server sẽ tự động tìm port khác
